---

Snippet Bubble Manager includes 5 built-in templates accessible from the New Snippet modal. Each template pre-fills the title, code, language, description, and tags fields so you can get started quickly without writing boilerplate from scratch.

## Accessing templates

To use a template:

<Steps>
  <Step title="Open the New Snippet modal">
    Click the **New Snippet** button in the sidebar or main toolbar.
  </Step>
  <Step title="Navigate to the AI generation section">
    Scroll to the **Generate with AI** section at the bottom of the snippet form.
  </Step>
  <Step title="Select a template">
    Click **Use Template** to open the template picker, then select the template you want.
  </Step>
  <Step title="Edit and save">
    All form fields are pre-filled from the template. Edit any field before clicking **Save**.
  </Step>
</Steps>

<Note>
  Applying a template fills in the title, code, language, description, and tags fields in the form. You can edit any of these values before saving — the template is just a starting point.
</Note>

<Tip>
  You can combine templates with AI generation: apply a template to get a working baseline, then use **Generate with AI** to modify or extend it based on a prompt.
</Tip>

    **Description:** A basic Express.js route handler with error handling.

    **Tags:** `express`, `node`, `api`

    ```typescript
    import { Request, Response } from 'express';

    export const getItems = async (req: Request, res: Response) => {
      try {
        const items = await db.collection('items').get();
        res.status(200).json({
          success: true,
          data: items
        });
      } catch (error) {
        res.status(500).json({
          success: false,
          message: 'Server Error'
        });
      }
    ```
