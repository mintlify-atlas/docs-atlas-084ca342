Snippet Bubble Manager integrates with Google Gemini to provide two AI-powered features: generating a complete snippet from a text description, and suggesting inline code completions while you write in the Monaco editor.

<Warning>
  Both AI features require a valid Gemini API key. Set the key as the `GEMINI_API_KEY` environment variable before starting the application. Requests will fail if this variable is missing or invalid.
</Warning>

## Generate a snippet from a prompt

The **Generate with AI** section appears inside the New Snippet modal. Use it to produce a complete snippet — title, language, and code — from a plain-text description.

<Steps>
  <Step title="Open the New Snippet form">
    Click **New Snippet** in the header to open the snippet creation modal.
  </Step>
  <Step title="Enter a prompt">
    Find the **Generate with AI** section in the form. Type a description of the snippet you want in the prompt field.

    <Tip>
      Try prompts like `React hook for local storage` or `Python script to scrape a website` to get well-structured, ready-to-use snippets.
    </Tip>
  </Step>
  <Step title="Run the generation">
    Press **Enter** or click the **Generate** button. The form populates the **Title**, **Code**, and **Language** fields automatically with the AI-generated output.
  </Step>
  <Step title="Review and save">
    Inspect the generated code in the Monaco editor. Edit any field as needed, then click **Save**.
  </Step>
</Steps>

<Note>
  The generator uses the `gemini-3-flash-preview` model, which is optimized for fast responses. It returns structured JSON with `code`, `language`, and `title` fields, ensuring the form fields are always populated correctly.
</Note>

## Inline code completions

While you write code in the Monaco editor (inside the snippet form), the editor automatically requests AI-powered completions from Gemini.

### How completions work

- Completions are triggered automatically when you type certain characters: `.`, ` ` (space), `(`, or `{`.
- At least 10 characters of existing code must be present before completions activate.
- Suggestions appear as completion items labeled **AI Suggestion — Generated by Gemini** in the editor's autocomplete dropdown.
- Accept a suggestion by selecting it from the list.

### Completion behavior

| Setting | Value |
|---|---|
| Model | `gemini-3-flash-preview` |
| Temperature | 0.2 (low, for more predictable output) |
| Max tokens | 100 |
| Trigger characters | `.` ` ` `(` `{` |

Completions are scoped to the currently selected language, so the model receives the correct language context with each request.
