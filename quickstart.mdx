<Warning>
  Snippet Bubble Manager requires a Firebase project with Firestore and Authentication enabled. Complete [Firebase Setup](/firebase-setup) before continuing.
</Warning>

## Prerequisites

- Node.js 18 or later
- A Firebase project ([set one up here](/firebase-setup))
- A Google account for sign-in

## Get up and running

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/CassieMarie0728/snippet-bubble-manager.git
    cd snippet-bubble-manager
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Configure environment variables">
    Create a `.env.local` file in the project root for your API keys:

    ```bash .env.local
    GEMINI_API_KEY=your_gemini_api_key_here
    APP_URL=http://localhost:3000
    ```

    `GEMINI_API_KEY` is only required if you want AI snippet generation and inline code completions. You can skip it to get the app running first. `APP_URL` should match the address where the dev server runs.
  </Step>

  <Step title="Add firebase-applet-config.json">
    Create `firebase-applet-config.json` in the project root. This file is the primary Firebase configuration — it is imported directly by `firebase.ts`:

    ```json firebase-applet-config.json
    {
      "apiKey": "your_api_key_here",
      "authDomain": "your_project.firebaseapp.com",
      "projectId": "your_project_id",
      "storageBucket": "your_project.appspot.com",
      "messagingSenderId": "your_sender_id",
      "appId": "your_app_id",
      "firestoreDatabaseId": "(default)"
    }
    ```

    Find these values in the Firebase console under **Project Settings → Your apps → SDK setup and configuration**. See [Firebase Setup](/firebase-setup) for the full walkthrough.

    <Note>
      Use `"(default)"` for `firestoreDatabaseId` unless you created a named Firestore database.
    </Note>
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>

  <Step title="Sign in with Google">
    On the landing screen, click **Sign in with Google**. Firebase Authentication handles the OAuth flow — your Google account becomes the owner of your snippet library.
  </Step>

  <Step title="Create your first snippet">
    1. Click **New Snippet** in the top-right header.
    2. Fill in a **Title** (e.g. "Fetch with error handling").
    3. Select a **Language** from the dropdown (e.g. JavaScript).
    4. Paste or type your code in the Monaco editor.
    5. Click **Save**.

    Your snippet appears in the main grid and is immediately available in the floating bubble.

    <Tip>
      Instead of typing code manually, try the AI generation shortcut: type a description in the "Generate with AI" field and press **Enter**. Gemini will generate the title, code, and language automatically. See [AI Generation](/features/ai-generation) for details.
    </Tip>
  </Step>
</Steps>
